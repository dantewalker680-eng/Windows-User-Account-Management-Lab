# -Windows-User-Account-Management-Lab# Windows User Account Management Lab

## Objective

Demonstrate the ability to create, manage, disable, enable, and remove local Windows user accounts in Windows 11.

---

## Environment

- Windows 11 Virtual Machine
- VirtualBox
- PowerShell
- Local Administrator Account

---

## Tasks Completed

### Create a Local User Account

Created a new local user account for testing and administration purposes.

### Verify User Creation

Confirmed the account was successfully created and visible in Windows.

### Disable a User Account

Disabled the test account and verified it could no longer be used.

### Enable a User Account

Re-enabled the account and confirmed it was active again.

### Remove a User Account

Deleted the test account after completing validation.

- Windows User Account Management
- Local User Administration
- Account Enable/Disable Procedures
- PowerShell Administration
- Windows Security Fundamentals
- Troubleshooting User Accounts

## Key Takeaways

This lab provided hands-on experience managing local user accounts in Windows 11. These tasks are commonly performed by Help Desk and IT Support professionals when creating accounts, troubleshooting login issues, managing permissions, and maintaining system security.
